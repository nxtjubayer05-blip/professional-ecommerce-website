# Professional E-Commerce Website

## একটি সম্পূর্ণ পেশাদার ই-কমার্স ওয়েবসাইট প্রজেক্ট

### প্রযুক্তি
- HTML5, CSS3, JavaScript (ES6)
- PHP 8+, MySQL Database
- Firebase Authentication & Firestore Database
- Payment Gateway Integration (bKash, Nagad, Rocket, Card)
- AJAX, Font Awesome Icons
- Mobile First & Responsive Design

### ফিচার
✅ Firebase Authentication (Google, Facebook, Email, Phone)
✅ Product Management System
✅ Shopping Cart & Checkout
✅ Payment Gateway Integration
✅ Download System
✅ Admin Dashboard
✅ Order Management
✅ User Management
✅ Banner Management
✅ Responsive Design (Mobile, Tablet, Desktop)
✅ SEO Friendly
✅ PWA Ready
✅ Security (CSRF, SQL Injection, XSS Protection)

### ফোল্ডার স্ট্রাকচার
```
├── index.html                 # হোম পেজ
├── product-details.html       # প্রোডাক্ট বিস্তারিত
├── order.html                 # অর্ডার ফর্ম
├── payment.html               # পেমেন্ট পেজ
├── admin/
│   ├── index.html             # অ্যাডমিন ড্যাশবোর্ড
│   ├── products.html          # প্রোডাক্ট ম্যানেজমেন্ট
│   ├── banners.html           # ব্যানার ম্যানেজমেন্ট
│   ├── orders.html            # অর্ডার ম্যানেজমেন্ট
│   ├── users.html             # ইউজার ম্যানেজমেন্ট
│   ├── payments.html          # পেমেন্ট সেটিংস
│   └── settings.html          # ওয়েবসাইট সেটিংস
├── css/
│   ├── style.css              # মেইন স্টাইল
│   ├── responsive.css         # রেসপন্সিভ ডিজাইন
│   └── admin-style.css        # অ্যাডমিন স্টাইল
├── js/
│   ├── main.js                # মেইন স্ক্রিপ্ট
│   ├── firebase-config.js     # Firebase কনফিগ
│   ├── cart.js                # কার্ট ফাংশনালিটি
│   ├── payment.js             # পেমেন্ট হ্যান্ডলার
│   └── admin.js               # অ্যাডমিন স্ক্রিপ্ট
├── backend/
│   ├── config/
│   │   └── database.php       # ডাটাবেস কনফিগ
│   ├── api/
│   │   ├── products.php       # প্রোডাক্ট API
│   │   ├── orders.php         # অর্ডার API
│   │   ├── payments.php       # পেমেন্ট API
│   │   └── users.php          # ইউজার API
│   └── handlers/
│       ├── payment-handler.php # পেমেন্ট হ্যান্ডলার
│       └── auth-handler.php    # অথেন্টিকেশন হ্যান্ডলার
└── firebase-functions/
    └── index.js               # Firebase ফাংশনস
```
